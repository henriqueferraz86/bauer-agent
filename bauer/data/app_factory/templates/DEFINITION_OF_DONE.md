# DEFINITION_OF_DONE.md

Uma tarefa só está pronta quando:

- [ ] Código foi implementado
- [ ] Código foi revisado pelo Agent
- [ ] Entrada de dados foi validada
- [ ] Erros foram tratados
- [ ] Logs úteis foram adicionados, quando necessário
- [ ] Teste mínimo foi criado ou justificativa documentada
- [ ] Documentação foi atualizada
- [ ] Nenhum secret foi exposto
- [ ] Como testar foi documentado
- [ ] Impacto foi registrado em `PROGRESS.md`

## Definition of Done da V1

A V1 só está pronta quando:

- [ ] Aplicação roda localmente
- [ ] Healthcheck funciona
- [ ] Primeira vertical slice funciona
- [ ] Testes mínimos existem
- [ ] README ensina a rodar
- [ ] `.env.example` existe
- [ ] Segurança básica foi revisada
- [ ] Deploy foi documentado
- [ ] Runbook existe
- [ ] Delivery Score foi gerado
